Синтаксис файла для загрузки данных на сервер

Для загрузки данных на сервер необходимо описать конфигурационный файл YAML (выбирается в поле "Файл") и если есть изображения (выбираются в поле "Изображения").

Основы YAML

Для передачи серверу данных мы используем конфигурационный файл в формате YAML.

Наш конфигурационный файл YAML состоит из словарей. Посмотрим как писать словари в YAML. Словарь представляется в виде ключ: значение (двоеточие должно сопровождаться пробелом).

В файле можно оставлять комментарии поставив знак #. Все что будет на строке после данного знака является комментарием и при обработке файла не учитывается.

# Запись одного образца
sample:
1:
roughness: 0.2
weight: 14.1736
image: "sample.png"
date_proc_streng: "2020-05-03"
sample_material_id: 1
sample_type_id: 1

Блоки

Файл конфигурации состоит из 6 блоков.

Название блока Описание Обязательный
sample_type Тип образца Да
sample_material Материал из которого изготовлен образец Да
modification Информация по упрочнению Нет
sample Информация по образцу Да
trials Информация по испытанию Да
received_values Информация по эксперементу Да

Ниже блока вводится id объекта. Ниже id вводятся названия аттрибутов и их значения. В примере ниже мы зададим 2 объекта для блока sample.

sample:
1: # sample_id = 1
method "азотирование"
weight: 14.1736
2: # sample_id = 2
weight: 14.276
image: "sample.png"

Тип данных аттрибута

Каждый аттрибут имеет свой тип данных.

Название типа Правило заполнения Пример
строка Пишется в кавычках " " "азотирование"
целое число Пишется без кавычек 60
число с плавающей точкой Пишется без кавычек. Разделитель целого - точка. 14.234
дата Пишется в кавычках. Формат записи - ГГГГ-мм-дд "2018-03-27"
картинка Пишется в кавычках. Указать имя файла. Имена файлов не должны совпадать. "sample_1.png"

Аттрибуты

sample_type
Название аттрибута Тип Описание Обязательный
name строка Название типа образца Да

sample_material
Название аттрибута Тип Описание Обязательный
name строка Материал из которого изготовлен образец Да
type строка Тип материала Нет
description строка Дополнительное описание Нет

modification
Название аттрибута Тип Описание Обязательный
method строка Способ упрочнения Нет
exp_param строка Параметры упрочнения Нет
depth_coating число с плавающей точкой Толщина упрочнения Нет
hardness_coating число с плавающей точкой Твердость упрочнения Нет
organization строка Организация, которая провела упрочнение Нет
struct_coating строка Состав покрытия Нет

sample
Название аттрибута Тип Описание Обязательный
date_proc_streng дата Дата процесса упрочнения Нет
roughness число с плавающей точкой Шероховатость поверхности Нет
sub_hardness число с плавающей точкой Твердость подложки Нет
weight строка Первоначальная масса образца Да
marking строка Маркировка образца Нет
description строка Дополнительное описание Нет
image картинка Внешний вид образца, начальный Нет
material_id число id объекта из блока sample_material Да
type_id число id объекта из блока type_material Да
modification_id число id объекта из блока modification Нет

trials
Название аттрибута Тип Описание Обязательный
size_particle число Размер частиц Нет
speed_collision число Скорость соударения Нет
add_param строка Дополнительные параметры частицы Нет
corner_collision число Угол соударения Нет
date_trials дата Дата проведения испытания Нет
date_end_trials дата Время окончания испытания Нет
type_particle строка Тип частицы Нет
description строка Дополнительное описание Нет
flow_type строка Тип потока Нет
chamber_pressure строка Давление в камере Нет
sample_temp строка Температура образца Нет
abrasive_type строка Тип абразива Нет
air_consumption строка Расход воздуха Нет
abrasive_consumption строка Расход абразива Нет
nozzle_diam строка Диаметр сопла Нет
distance_sub строка Расстояние от сопла до образца Нет
sample_id число id объекта из блока sample Да

received_values
Название аттрибута Тип Описание Обязательный
change_weight число с плавающей точкой Масса образца после эксперимента Да
time_trials число Длительность эксперимента Да
image картинка Изображение образца после эксперимента Нет
trials_id число id объекта из блока trials Да
